Инструмент предназначен для запуска сеанса с гибким набором параметров.

Поля формы

  • Строка соединения ИБ - содержит информацию о способе подключения к базе и путь к ней
    • Для файловой базы: File="<путь к папке ИБ>";
    • Для клиент-серверной базы: Srvr="<имя кластера серверов>"; Ref="<имя ИБ в кластере>";
    • При подключении через веб-сервер: ws="<URL>";
  • Режим запуска:
    • Тонкий клиент - выбран по умолчанию, тонкий клиент управляемого приложения,
    • Толстый клиент УФ - толстый клиент управляемого приложения,
    • Обычное приложение - толстый клиент обычного приложения,
    • Конфигуратор - сеанс откроется в режиме Конфигуратор.
  • Параметр запуска - параметр запуска, передаваемый через ключ командной строки /C. В списке выбора добавлены параметры запуска, которые использует БСП;
  • Код разрешения - соответствует ключу командной строки /UC. Позволяет запустить сеанс при установленной блокировке начала сеансов;
  • Завершить текущий сеанс - если установлено, после нажатия кнопки "Запустить сеанс" текущий будет сразу завершен
  • Закрыть эту форму - если установлено, после нажатия кнопки "Запустить сеанс" форма инструмента Запуск сеанса будет закрыта
  • Открыть Инструментики при запуске - если установлено, в новом сеансе автоматически откроются Инструментики. Доступно только при установленном режиме запуска "Тонкий клиент" или "Толстый клиент УФ". Также, данная функция недоступна, если текущий сеанс открыт через веб-клиент,
  • Использовать аутентификацию ОС - 1С попытается авторизоваться, используя службу Active Directory. Если это не удастся, пользователю будет предложено ввести имя пользователя и пароль;
  • Имя пользователя - имя пользователя, которое будет передано при открытии клиента;
  • Пароль - пароль, который будет передан при открытии клиента.
Важно: установка параметра "Пароль" может привести к его компрометации. Это связано с тем, что пароль будет передан в качестве значения параметра командной строки.
Любой пользователь или программа ОС, имеющий доступ к списку процессов текущего пользователя, сможет увидеть пароль в открытом виде.
Не рекомендуется использовать данный параметр на "боевых" информационных базах, особенно на серверах удаленных рабочих столов.

Как это работает?

Инструмент запускает клиент 1С:Предприятия с соответствующими параметрами. Использование параметров соответствует Приложению 7 Руководства администратора для платформы 8.3.23 ("Параметры командной строки запуска «1С:Предприятия»")
