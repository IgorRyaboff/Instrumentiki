Консоль WebSocket

Консоль предназначена для отладки взаимодействия с WebSocket-серверами. Для использования инструмента требуется технологическая платформа версии 8.3.27 или выше.

Установка соединения

Для подключения к WebSocket-серверу введите параметры подключения, (пример - на скриншоте ниже). Для подключения по незащищенному WebSocket используйте протокол "ws://", для подключения к защищенному WebSocket - протокол "wss://".

Указание адреса сервера и таймаута является обязательным. Также, опционально можно указать логин и пароль (для проведения базовой HTTP-аутентификации при установке соединения), а также дополнительные HTTP-заголовки.
Для подключения следует использовать кнопку "Установить соединение" в верхнем левом углу формы.

Прием и отправка сообщений

С момента нажатия кнопки "Установить соединение" начнут появляться сообщения в списке:

Для отправки текстового сообщения WebSocket-серверу используйте кнопку "Отправить текст...". Для отправки двоичных данных - "Отправить файл...".
Для очистки сообщений можно использовать кнопку "Очистить сообщения" (нажатие этой кнопки не приводит к закрытию соединения).

Пиктограммы в списке сообщений означают:
  •  - входящее сообщение от сервера;
  •  - исходящее сообщение от клиента;
  •  - служебное сообщение о статусе соединения (при установке и разрыве соединения, а также при возникновении ошибок).
