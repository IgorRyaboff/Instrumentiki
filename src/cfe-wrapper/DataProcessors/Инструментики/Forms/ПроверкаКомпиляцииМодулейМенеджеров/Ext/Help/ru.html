Назначение инструмента

Инструмент предназначен для проверки наличия модулей менеджеров, разработанных без вставки директив "#Если Сервер Или ТолстыйКлиентОбычноеПриложение Или ВнешнееСоединение Тогда".
Такие модули могут не выполняться в режиме толстого клиента управляемого приложения.
Подробнее на ИТС: Поддержка толстого клиента, управляемое приложение, клиент-сервер

Использование инструмента

Для использования инструмента необходимо запустить программу в режиме толстого клиента управляемого приложения.
Для выполнения проверки следует нажать "Выполнить проверку".

При возникновении ошибки проверка будет прервана платформой. При этом в форме ошибки не будет кнопки закрытия формы, только "Завершить работу" и "Перезапустить". То есть, все несохраненные в текущем сеансе данные будут утеряны.
