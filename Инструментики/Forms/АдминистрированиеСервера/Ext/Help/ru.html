Назначение инструмента

Инструмент предназначен для взаимодействия с кластером серверов 1С:Предприятия.

Инструмент поддерживает два метода подключения:
  • COM-соединение - требует зарегистрированной компоненты V83.ComConnector из дистрибутива текущей версии платформы. Работает только на серверах с операционной системой Windows;
  • Сервер администрирования (RAS) - работает на любой ОС сервера, но требует запуска сервера администрирования (ras.exe).


Вкладка "Информационные базы"

Показывает список зарегистрированных в кластере информационных баз.

Создание новой информационной базы

По нажатию кнопки "Создать" открывается форма создания информационной базы.
Для создания информационной базы копированием выберите копируемую базу в списке и нажмите F9 (либо нажмите по строке правой кнопкой мыши и выберите "Скопировать".

Изменение параметров существующей информационной базы

При выборе существующей ИБ в списке запрашивается информация об этой базе, после чего открывается форма редактирования параметров этой базы.

Удаление информационной базы

Для удаления ИБ используется команда "Удалить" из контекстного меню.
При удалении необходимо указать логин и пароль администратора ИБ, вручную ввести имя информационной базы (для контроля правильности выбора базы в списке) и выбрать режим удаления:
  • Оставить базу данных без изменений
  • Очистить базу данных
  • Удалить базу данных
