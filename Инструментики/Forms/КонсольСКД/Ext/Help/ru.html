Данный инструмент позволяет проводить отладку схем компоновки данных.

В командной панели расположены следующие команды:
  • Сформировать - выполняет компоновку схемы компоновки данных с текущими настройками;
  • Загрузить схему - вызывает диалог загрузки схемы компоновки данных в инструмент;
  • Открыть настройки - открывает форму настроек отчета;
  • Конструктор СКД - открывает текущую схему компоновки данных в конструкторе. Команда доступна только в режиме "Толстый клиент".

Вкладка "Отчет"

Содержит табличный документ, в который выводится результат компоновки, выполняемой командой "Сформировать".

Вкладка "Наборы данных макета"

Выводит информацию о наборах данных, сформированных компоновщиком макета компоновки данных.
После передачи схемы компоновки данных и настроек компоновки данных компоновщик генерирует макет, в котором содержатся измененные под текущие настройки наборы данных. При компоновке СКД использует именно те наборы данных, которые выводятся во вкладке "Наборы данных макета".
Тексты запросов наборов данных с типом Запрос можно открыть в Консоли запросов, воспользовавшись гиперссылкой "Открыть в Консоли запросов".

Вкладка "Значения параметров макета"

Содержит имена и значения параметров, которые подставляются в наборы данных "Запрос" наборов данных, сгенерированных компоновщиком макета (эти наборы данных описаны во вкладке "Наборы данных макета").
