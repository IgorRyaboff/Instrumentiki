Инструмент предназначен для получения информации из представления "битой ссылки".
Пример такой ссылки: <Объект не найден> (237:a5e2f93914fe6d674e31efd39e4f7b59)

Для использования инструмента необходимо вставить ссылку в поле "Текст" и нажать Расшифровать. Из ссылки будут извлечены следующие данные:
- Объект метаданных - полное имя объекта метаданных, с которым связана ссылка;
- Идентификатор объекта - уникальный идентификатор объекта;
- Ссылка - полученная в процессе выполнения ссылка на объект. Если объект отсутствует в текущей ИБ, представление ссылки повторит введенный текст;
- Навигационная ссылка - навигационная ссылка, по которой можно перейти, используя комбинацию SHIFT+F11.
